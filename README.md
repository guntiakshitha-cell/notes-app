# notes-app
Its a notes app where users can write notes, edit them ,save them and delete them if they want and they are stored in the mongo db database


Tech Stack
- Node.js
- Express.js
- MongoDB
- HTML
- CSS
- JavaScript

Features
- Add Notes
- Edit Notes
- Delete Notes
- Store notes in MongoDB

How to Run

1. npm install
2. node server.js
3. open http://localhost:5000
